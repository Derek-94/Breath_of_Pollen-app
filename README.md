# 花粉の呼吸 (Pollen Breathing)

일본·한국 꽃가루 & 날씨 정보 앱 (iOS)

## 주요 기능

- 오늘의 꽃가루 레벨 및 시간별 예보
- 주간 날씨 & 꽃가루 예보
- 옷차림 추천 및 빨래 지수
- UV, PM2.5, 습도 정보
- 저녁 꽃가루 알림 (시간 설정 가능)
- 다크모드 지원
- 다국어 지원 (한국어 / 일본어 / 영어)

## 기술 스택

- Expo SDK 55 / React Native 0.83.4
- TypeScript
- Expo Router v6 (파일 기반 라우팅)
- Expo Notifications

## 실행 방법

```bash
npm install
npx expo run:ios
```
